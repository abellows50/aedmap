API Reasorces

The website supports a responsive API by http request, no api key is needed. Instructions are bellow.

Description:

This API makes requests to the server to get the aed's in a specific radius that are in our data base.


Python Syntex

After downloading the file linked bellow (api.py) use this code to make requests

import aed-api as aed
#this initializes the api request object
myAedApi = aed.ApiRequest()

#make a request
myAedApi.request(lat,long,radius,format) #returns the response value and saves it in myAedApi.response

The ApiRequest object has a few internal values that you are able to access:

,
Api PropertyWhat it does
.latthe lat of our last request
.lngthe lng of your last request
.rthe radius of your last request
.formatthe format of your last request
.responsethe server response
.errorFalse if no error True if an error occured
.errorTextThe text of the server error response
